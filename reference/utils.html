cana.utils.expand_logic_line(line)[source]¶

This generator expands a logic line containing - (ie. 00- 0 or 0-0 1) to a series of logic lines containing only 0 and 1.

Parameters:line (string) – The logic line. Format is <binary-state><space><output>.
Returns:a series of logic lines
Return type:generator

Example

>>> expand_logic_line('1-- 0')
>>> 100 0
>>> 101 0
>>> 110 0
>>> 111 0
