Notation¶

This notation follows the initial notation in [MPR13] from where all subsequent papers derive.

Node (automata)¶

symbol description
\(x\)

Boolean Automaton (BooleanNode)

Binary-state automaton
\(k\)

Number of inputs of \(x\)

How many inputs determine the transitions of an automaton \(x\)
\(F\)

Look-up table (LUT) of \(x\)

The transition function of \(x\) represented as a LUT (\(2^k\) entries)
\(f_\alpha\)

LUT entry in \(F\)

k-tuple combination of input states i.e. condition and corresponding transition
\(c_{\alpha}\)

condition part in a LUT entry of \(f_\alpha\)

k-tuple combination of input states i.e. condition
\(s_{\alpha}\)

transition in a LUT entry \(f_\alpha\)

Boolean state prescribed as the transition in \(f_\alpha\)

Network¶

symbol description
\(B\)

Boolean Network (BooleanNetwork)

A graph of \(n\) automata with directed edges (source node is input of end node)
\(X\)

set of automata in \(B\)

set of Boolean automata that constitute a Boolean Network \(B\)
\(n\)

number of nodes in \(B\)

\(n = |X|\)
\(\textbf{x}\)

network configuration

collection of the states of all nodes in a Boolean Networ \(B\)
\(X_i\)

set of inputs of nodes \(X_i\) in \(B\)

Set of input nodes of \(X_i\)
\(k_i\)

in-degree of \(X_i\)

Cardinality of \(X_i\)
\(F_i\)

Look-up table of \(X_i\)

Transition function represented as a LUT.
\(f_{i:\alpha}\)

LUT entry in \(F_i\)

Sub-indices \(i\) and \(\alpha\), separated by ’:’, are used to specify node and entry.
\(A_i\)

An attractor of \(B\)

A specific (index \(i\)) fixes-point or periodic attractor of a B.N. \(B\)
\(\sigma(\textbf{x}) \rightarrow A\)

Dynamic trajectory of \(\textbf{x}\) to \(A\)

This notation is used to represent that the trajectory of some configuration \(\textbf{x}\) is known to converge to \(A\)

Wildcard¶

symbol description
\(\#\)

Wildcard symbol

If this symbol appears in a condition, the variable it represents can be in any state
\(F^{'}\)

wildcard-schema redescription of \(F\)

LUT where entries are wildcard schemata
\(f^{'}_{v}\)

a wildcard schema in \(F^{'}\)

An entry in \(F^{'}\) is like an entry in \(F\) but its condition part can have wildcard symbols
\(\Upsilon_{v}\)

Entries \(f_{\alpha} \in F\) in \(f^{'}_{v}\)

collection of the states of all nodes in a Boolean Network \(B\)

Two-Symbol¶

symbol description
\(\circ_m\)

Position-free symbol

If a variable in the condition part of a schema is marked with this symbol, it can exchange places with any other variable in the same schema marked with the same symbol. Index \(m\) is used to distinguish subsets of identically- marked inputs.
\(\beta\)

Depth of search for two-symbol schemata

Defines the minimum number of wildcard schemata in a two-symbol redescription.
\(F''\)

2-Symbol redescription of \(F\).

LUT where entries are two-symbol schemata.
\(f''_{\theta}\)

a 2-Symbol schema in \(F''\)

An entry in \(F''\) is like an entry in \(|F|\) but its condition part can have wildcard and position-free symbols.
\(\Theta_{\theta}\)

Entries \(f_{\alpha} \in F : f'_v \rightarrow F''_{\theta}\)

The set of original LUT entries in \(F\) redescribed by a single 2-symbol schema \(\Theta'_{\theta} = \{ f'_{\alpha} : f_{\alpha} \rightarrow f''_{\theta} \}\)
\(\Theta'_{\theta}\)

Schemata \(f'_v \in F' : f'_v \rightarrow f''_{\theta}\)

The set of wildcard schemata in F’ redescribed by a single 2-symbol schema \(\Theta'_{\theta} = \{ f'_v : f_v \rightarrow f''_{\theta} \}\).
\(X_l\)

set of liberals enputs in a schema \(f''\)

The variables in the condition part of schema \(f''\) that are specified in a Boolean state (not wildcard)
\(\eta_l\)

size of literal-emput set in schema \(f''\).

\(\eta_{l} = |X_{l}|\)
\(X_l^s\)

\(\text{state-}s\) literal enputs in \(f''\)

Subset \(X_l^s \subset X_l\) of literal enput in a scpecific state \(s : s \in {0,1}\).
\(X_g\)

group invariant enput in a schema \(f''\)

The set variables in the condition part of schema \(f''\) that are marked with an identical position-free symbol, in every state they can take.
\(X_g^s\)

elements of \(X_g\) in state \(s\)

This notation is used to refer to the members of a group-invariant enput instantiated in a specific state \(s\), that is \(X_g^s = { \forall x_i \in X_g \land x_i = s}\).
\(\eta\)

number of group-invariant enputs in \(f''\)

Number of subsets of inputs marked with a distinct position-free symbol.
\(n_g\)

size of a single group-invariant enput \(g\) in \(f''\)

Number of inputs marked with the position-free symbol in \(g\).
\(n_g^s\)

a sub-constraint in \(X_g\) on state \(s \in {0,1}\)

specifies a group-invariant constraint in the set \(X_g\), at least \(n_g^s\) variables must be in state \(s\).
